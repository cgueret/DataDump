Welcome

Welcome to the website of the Dutch Census. This website provides access to all tables of the census published between 1795 and 1971. Additionally, most explanations, introductions and appendices will be available digitally soon (see meanwhile the legacy documentation). In total, more than 40,000 pages, originally published in more than 200 books, areâ€‹accessible. Three projects have been working in the digitization of the censuses since 1997.

The census tables are available in the following formats:

  • MS Excel spreadsheets
  • MS Excel spreadsheets with TabLinker markup
  • Comma-separated Values (CSV)
  • RDF, following the RDF Data Cube vocabulary schema

Previous releases

This is a refactored version of the volkstellingen.nl) website. The migration of some contents is still pendant:

  • Documentation pages
  • Images of scanned materials

The authoritative source for data from the 1795-1971 censuses is the online archiving system EASY of DANS.

Download

To download the full dataset, click one of the following links:

  • Tarball
  • Zip

To download a file of a specific type, click one of the following links:

  • Original raw Excel spreadsheets
  • Original Excel with markup
  • CSV conversion of the spreadsheets
  • Conversion of the previous directory to RDF Turtle, Bzip2 compressed
  • Legacy HASH project database

A list of the files of the chosen type will appear in the GitHub repository. Click on the desired file, then click on the Raw button to download it.
