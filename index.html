Welcome

Welcome to the website of the Dutch Census. This website provides access to all tables of the census published between 1795 and 1971. Additionally, most explanations, introductions and appendices will be available digitally soon (see meanwhile the legacy documentation). In total, more than 40,000 pages , originally published in more than 200 bands , made ​​accessible . At the digitalisation of censuses worked in three projects since 1997.   Copy archive site

Because the underlying CMS could not be supported DANCE was forced to set up the censuses on line available . Alternative way This website is the result . All census records are always available and downloadable , including scans. Although the full-text search functionality is no longer available , you can still utilize the functionality under 'Find Systematic ' . If you have questions or comments about this modified form of the census website , please email us at volkstellingen@dans.knaw.nl .   " Two centuries counted " is available

The book Two centuries counted . Research with the digital public , vocational, and housing censuses 1795-2001 can be ordered using the order form on the website DANCE .  Image:vt_bundel download data

The data from the 1795-1971 censuses are also available through the online archiving system EASY of DANS .

Welcome to GitHub Pages.

This automatic page generator is the easiest way to create beautiful pages for all of your projects. Author your page content here using GitHub Flavored Markdown, select a template crafted by a designer, and publish. After your page is generated, you can check out the new branch:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Designer Templates

We've crafted some handsome templates for you to use. Go ahead and continue to layouts to browse through them. You can easily go back to edit your page before publishing. After publishing your page, you can revisit the page generator and switch to another theme. Your Page content will be preserved if it remained markdown format.

Rather Drive Stick?

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
